🔥 OpenClaw 简介

OpenClaw 是一个自托管的 AI Agent 网关，让你可以在自己的机器上运行 AI 助手，并通过多种聊天应用（WhatsApp、Telegram、Discord、iMessage 等）与之交互。

为什么选择 OpenClaw？

  • 完全自托管 — 数据在自己机器上，不用依赖第三方云服务
  • 多平台统一 — 一个 Gateway 同时服务多个聊天应用
  • 多 Agent 隔离 — 工作/生活场景完全分离
  • 移动端扩展 — iOS/Android 节点可调用相机、屏幕、位置等硬件
  • 开放生态 — Skills 系统可扩展，有丰富的插件市场
核心架构：Gateway 作为中心枢纽，连接各种聊天渠道和 AI Agent，同时支持移动端节点扩展能力。

📦 安装与配置

快速安装

# 安装 OpenClaw
npm install -g openclaw@latest

# 运行配置向导
openclaw onboard --install-daemon

# 登录 WhatsApp（或其他渠道）
openclaw channels login

# 启动 Gateway
openclaw gateway --port 18789

配置文件位置

文件 路径 说明
主配置 ~/.openclaw/openclaw.json Gateway 核心配置
工作区 ~/.openclaw/workspace Agent 文件和记忆
Sessions ~/.openclaw/agents/<agentId>/sessions 会话历史
Skills ~/.openclaw/skills 共享技能

🧠 多 Agent 路由

多 Agent 是 OpenClaw 最强大的功能之一。每个 Agent 有独立的工作区、配置和会话。

配置示例

{
  "agents": {
    "list": [
      {
        "id": "chat",
        "name": "日常聊天",
        "workspace": "~/.openclaw/workspace-chat",
        "model": "anthropic/claude-sonnet-4-5"
      },
      {
        "id": "opus",
        "name": "深度工作",
        "workspace": "~/.openclaw/workspace-opus",
        "model": "anthropic/claude-opus-4-6"
      }
    ]
  },
  "bindings": [
    { "agentId": "chat", "match": { "channel": "whatsapp" } },
    { "agentId": "opus", "match": { "channel": "telegram" } }
  ]
}

路由优先级（从高到低）

  1. 特定用户/群组 (peer)
  2. 父级会话继承 (parentPeer)
  3. Discord 角色/服务器
  4. 账号级别 (accountId)
  5. 频道级别
  6. 默认 Agent
提示：使用 openclaw agents add <name> 快速创建新 Agent，使用 openclaw agents list --bindings 查看路由配置。

📱 移动端节点

将 iPhone/Android 设备配对到 Gateway，让 AI 获得移动端的硬件能力。

常用命令

# 查看节点状态
openclaw nodes status

# 拍照（前后摄像头）
openclaw nodes camera snap --node my-iphone --facing front

# 录屏
openclaw nodes screen record --node my-iphone --duration 10s

# 获取位置
openclaw nodes location get --node my-iphone --accuracy precise

# 在手机上展示网页
openclaw nodes canvas present --node my-iphone --target https://example.com

配对流程

  1. 在手机上安装 OpenClaw App
  2. 连接到 Gateway WebSocket
  3. 在 Gateway 上批准配对：openclaw devices approve <requestId>
  4. 开始使用节点功能
注意：节点 App 需要保持在前台才能使用 canvas.* 和 camera.* 功能。

🛠️ Skills 系统

Skills 是 OpenClaw 的扩展机制，类似插件。可以从 ClawHub 安装或自己开发。

安装 Skill

# 从 ClawHub 安装
clawhub install gh-issues
clawhub install weather
clawhub install video-frames

# 更新所有 Skills
clawhub update --all

Skill 优先级

  1. Workspace skills (<workspace>/skills) — 最高
  2. 本地 skills (~/.openclaw/skills)
  3. 内置 bundled skills — 最低

配置 Skill

{
  "skills": {
    "entries": {
      "nano-banana-pro": {
        "enabled": true,
        "apiKey": { "source": "env", "provider": "default", "id": "GEMINI_API_KEY" },
        "env": {
          "GEMINI_API_KEY": "your-key-here"
        }
      }
    }
  }
}

🖥️ Control UI 使用

Control UI 是 OpenClaw 的浏览器控制台，提供图形化界面管理 Gateway。

访问方式

  • 本地：http://127.0.0.1:18789/
  • Tailscale：https://<magic-dns>/

主要功能

  • 实时聊天 + 查看工具调用
  • 管理多个 Channel（WhatsApp/Telegram/Discord 等）
  • 查看和管理 Sessions
  • 配置 Cron 定时任务
  • 管理 Skills
  • 查看日志和调试信息
远程访问建议：使用 openclaw gateway --tailscale serve 通过 Tailscale 安全访问 Control UI。

⏰ 定时任务

配置 Cron 任务让 Agent 定时执行操作。

配置示例

{
  "cron": {
    "jobs": [
      {
        "name": "每日天气",
        "schedule": "0 8 * * *",
        "prompt": "查一下今天的天气并告诉我",
        "target": "telegram",
        "channel": "telegram"
      }
    ]
  }
}

常用 Cron 表达式

表达式 含义
0 8 * * * 每天早上 8 点
0 */6 * * * 每 6 小时
0 0 * * 1 每周一凌晨

🔒 沙盒与安全

为不同 Agent 设置不同的安全级别，隔离风险。

配置示例

{
  "agents": {
    "list": [
      {
        "id": "personal",
        "sandbox": { "mode": "off" }
      },
      {
        "id": "family",
        "sandbox": {
          "mode": "all",
          "scope": "agent"
        },
        "tools": {
          "allow": ["read", "exec"],
          "deny": ["write", "edit", "browser"]
        }
      }
    ]
  }
}

沙盒模式

  • off — 无沙盒，完全信任
  • all — 始终沙盒
  • ask — 询问用户

❓ 常见问题

如何查看所有 Session？

openclaw sessions list
openclaw sessions history <session-key>

如何向其他 Session 发送消息？

openclaw sessions send --session <key> "消息内容"

如何生成子 Agent 处理任务？

在对话中输入：/spawn 任务描述

如何重启 Gateway？

openclaw gateway restart

如何查看日志？

openclaw logs tail
